Программа сделана с использование J2EE, поэтому
для установки необходимо использовать сервер приложений TomEE.

Для установки необходимы сборки TomEE:
apache-tomee-plus или apache-tomee-plume версии 7.0.2 и выше
Скачать можно отсюда: http://openejb.apache.org/downloads.html

Установка TomEE:
1. Распаковать скачанный архив в любую папку на диске.
2. Создать внутри нее каталог apps (к примеру должно быть так E:\apache-tomee-plus-7.0.2\apps)
3.В каталоге conf, который внутри папки сервера найти файл tomee.xml и раскомментировать в нем:
<Deployments dir="apps" /> (убрать окружающие скобки <!-- и --> )
4. В каталоге conf найти файл system.properties и закомментировать
строчку tomee.serialization.class.blacklist = * поставив перед ней значок #

Установка приложения:
1. Для установки приложения необходимо в установочном файле deploy.bat
поправить путь установки TomEE в переменной SERVERPATH

2. Для установки программы в системе уже должен быть установлен сборщик Maven (http://maven.apache.org/download.cgi)

3. Установка прогаммы и запуск сервера: deploy.bat

Запуск приложения:
После установки приложение доступно по ссылке:
http://127.0.0.1:8080/app/
